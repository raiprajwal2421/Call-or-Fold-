# Call or Fold?(Summer Project By SciMathSoc)
This repository contains all the relevant project  resources and assignments(along with few optional ones) for the Summer Project.
This project has been divided into following segements:-
- Probability
- Game Theory\
The end goal of the project is to build a simple poker simulator based on Game Theory Optimal(GTO) and Counterfactual Regret Minimization(CFR) that makes decision in real-time that could resemble a real poker agent as much as possible 
## Instructions for Submission:-

* Every mentee is supposed to create a folder for their assignments submission(in the main branch itself)
* Format for naming the folder :- **Roll-No_Name_Call_Or_Fold**
* Upload your assignments in your respective folder
